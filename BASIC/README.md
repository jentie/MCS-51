## Example Commands

|                  |                        |
|------------------|------------------------|
| print mtop       | size of RAM            |  
| print "hallo"    | print string           |
| 10 print "hallo" | program line 10        |
| 20 goto 10       | program line 20        |
| run              | run program            |
| ^c / Ctrl-C      | stop program execution |
| list             | show program           |
| new              | clear program space    |
