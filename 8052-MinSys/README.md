# 8052-MinSys - 8052 minimal system

Board with test sockets for 8051-family controller and 32K RAM

Features:
* 3 LED on Port1, Bit0 ... Bit2, active low
* 1 input key, Port1, Bit3
* 6pin ISP interface
* USB module with power and Rx/Tx
* 11.0592 MHz oszillator
* RESET key
