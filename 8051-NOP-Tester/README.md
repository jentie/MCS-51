# 8051 NOP Tester

typical simple CPU tester

## characteristics
* MCS-51 uses multiplexed address bus - AD0..AD7
* command data during special phase
* NOP command is 0x00
* MCS-51 controller can include ROM part
* input /EA tied to GND to disable internal ROM
